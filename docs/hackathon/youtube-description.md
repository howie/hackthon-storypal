# YouTube Description – StoryPal Hackathon Entry

## English Version

StoryPal – AI-Powered Interactive Storytelling for Kids | Gemini Live Agent Challenge

StoryPal is an AI-powered interactive storytelling and tutoring platform designed for children ages 3–8. Built with Google Gemini's multimodal capabilities — including LLM, TTS, Imagen, and the Live API — StoryPal creates engaging, personalized learning experiences through voice-driven stories, interactive games, and an AI tutor.

🌟 Key Features:
• Voice Story – AI crafts personalized interactive stories with multi-character voices and beautiful illustrations
• Voice Game – Children make choices in the story world for an immersive interactive adventure
• AI Tutor – An AI tutor that answers all kinds of curious questions in a child-friendly way

🛠️ Tech Stack:
• Backend: Python, FastAPI, SQLAlchemy, PostgreSQL
• Frontend: TypeScript, React, Vite, Tailwind CSS
• AI: Google Gemini (LLM, TTS, Imagen, Live API)
• Infrastructure: Terraform (GCP), Docker

This video was created for the purposes of entering the Gemini Live Agent Challenge hackathon.

## 繁體中文版

StoryPal – AI 互動說故事平台，專為兒童打造 | Gemini Live Agent Challenge

StoryPal 是一款專為 3–8 歲兒童設計的 AI 互動說故事與學習平台。運用 Google Gemini 的多模態能力 — 包含 LLM、TTS、Imagen 與 Live API — 打造沉浸式的個人化學習體驗，涵蓋語音故事、互動遊戲與 AI 家教。

🌟 核心功能：
• 語音故事 – AI 即時生成個人化互動故事，搭配多角色配音與精美插圖
• 語音遊戲 – 孩子在故事世界中做出選擇，體驗沉浸式互動冒險
• AI 家教 – AI 家教以孩子聽得懂的方式，回答各種好奇問題

🛠️ 技術架構：
• 後端：Python、FastAPI、SQLAlchemy、PostgreSQL
• 前端：TypeScript、React、Vite、Tailwind CSS
• AI：Google Gemini（LLM、TTS、Imagen、Live API）
• 基礎設施：Terraform（GCP）、Docker
